使用Admin站点（管理后台）

  • 一个网站通常包含前台和管理后台两部分
  • Django默认提供了管理后台，不需要再自己开发
  • 管理后台实现对网站的数据进行增删改查管理
  • 通过配置，可以实现django管理后台不同的展示样式
  • 使用
    1. 配置语言和时区
    2. 创建管理员账号
    3. 注册模型类
    4. 自定义管理后台界面显示

使用管理后台

  1. 配置语言和时区：修改settings.py文件

     LANGUAGE_CODE = 'zh-hans'    # 指定语言为中文（注意不要写错）
 TIME_ZONE = 'Asia/Shanghai'  # 指定时区 亚洲/上海

  2. 创建登录后台的管理员

     python manage.py createsuperuser

    需要指定： 用户名，邮箱，密码（至少8位）

  3. 注册模型类：在应用下的admin.py中注册模型类，注册后才会在管理后台显示

     # users/admin.py:
 from app01.models import Department, Employee

 # 注册Model类
 admin.site.register(Department)
 admin.site.register(Employee)

  4. 启动服务器，访问 http://127.0.0.1:8000/admin 地址，进入管理后台

  5. 通过管理后台界面，可实现对数据的增删改等管理操作
