# if-else

>想一想：在使用if的时候，它只能做到满足条件时要做的事情。那万一需要在不满足条件的时候，做某些事，该怎么办呢？
 
>答：使用 if-else

## <1>if-else的使用格式

```python
	if 条件:
		满足条件时要做的事情1
		满足条件时要做的事情2
		满足条件时要做的事情3
		...(省略)...
	else:
		不满足条件时要做的事情1
		不满足条件时要做的事情2
		不满足条件时要做的事情3
		...(省略)...
```

demo1

```python
	chePiao = 1 # 用1代表有车票，0代表没有车票
	if chePiao == 1:
		print("有车票，可以上火车")
		print("终于可以见到Ta了，美滋滋~~~")
	else:
		print("没有车票，不能上车")
		print("亲爱的，那就下次见了"）
```

结果1：有车票的情况

```python
	有车票，可以上火车
	终于可以见到Ta了，美滋滋~~~
```
结果2：没有车票的情况

```python
	没有车票，不能上车
	亲爱的，那就下次见了
```
