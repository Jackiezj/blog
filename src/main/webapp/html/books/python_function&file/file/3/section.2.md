# 字符串输出

demo

```python
	name = 'itheima'
	position = '讲师'
	address = '北京市'

	print('--------------------------------------------------')
	print("姓名：%s" % name)
	print("职位：%s" % position)
	print("公司地址：%s" % address)
	print('--------------------------------------------------')

```

结果:

```
	--------------------------------------------------
	姓名： itheima
	职位： 讲师
	公司地址： 北京市
	--------------------------------------------------
```
